--- title: An Impact Study of Program Data Shift Towards Code Analysis Model and Uncertainty Measurement layout: default ---

Description

We systematically study the effect of three common real-world program data shifts on DL software performance. We also investigate the effectiveness of existing state-of-the-artuncertainty approaches on programming tasks under data shift and propose a large-scale benchmark for their performance.
